# heanbian-util

## 前提条件

JDK11+

## pom.xml

具体版本，可以从 Maven Central 获得

```
<dependency>
  <groupId>com.heanbian.block</groupId>
  <artifactId>heanbian-util</artifactId>
  <version>1.0.3</version>
</dependency>
```
